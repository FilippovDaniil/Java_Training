# Модуль 02. Работа с переменными. Типы `int` и `String`

## Что такое переменная

**Переменная** — это именованная ячейка памяти, в которой хранится значение. У переменной есть **тип** (что хранит) и **имя** (как обращаться).

```java
int age = 25;        // тип int, имя age, значение 25
String name = "Иван"; // тип String, имя name, значение "Иван"
```

Схема в памяти:

```
  имя       тип       значение
 +------+ +------+  +--------+
 | age  | | int  |  |   25   |
 +------+ +------+  +--------+
```

---

## Объявление и инициализация

```java
int x;          // объявление (тип + имя), значения ещё нет
x = 10;         // инициализация (присваивание значения)

int y = 20;     // объявление + инициализация в одной строке
```

> Локальную переменную **нельзя использовать до присваивания** — компилятор выдаст ошибку.

---

## Тип `int` — целые числа

`int` хранит целые числа в диапазоне примерно от **−2,1 млрд до +2,1 млрд**.

```java
int a = 7;
int b = 3;

int sum  = a + b;   // 10  — сложение
int diff = a - b;   // 4   — вычитание
int mul  = a * b;   // 21  — умножение
int div  = a / b;   // 2   — ЦЕЛОЧИСЛЕННОЕ деление (дробная часть отбрасывается!)
int mod  = a % b;   // 1   — остаток от деления
```

> ⚠️ **Целочисленное деление:** `7 / 3` даёт `2`, а не `2.33`. Остаток получают через `%`.

---

## Тип `String` — строки

`String` хранит текст — последовательность символов в двойных кавычках.

```java
String greeting = "Привет";
String name = "Мир";
```

### Конкатенация (склейка строк) — оператор `+`

```java
String full = greeting + ", " + name + "!";  // "Привет, Мир!"
```

Если в `+` участвует строка, остальные значения **автоматически превращаются в текст**:

```java
int age = 25;
String s = "Возраст: " + age;   // "Возраст: 25"  (int стал текстом)
```

> ⚠️ Порядок важен из-за смешивания чисел и строк:
> ```java
> System.out.println(1 + 2 + "x");   // "3x"  — сначала 1+2=3, потом склейка
> System.out.println("x" + 1 + 2);   // "x12" — слева строка, всё склеивается
> ```

## Правила именования переменных

| Правило | Пример |
|---------|--------|
| начинается с буквы, `_` или `$` | `count`, `_tmp` |
| не может начинаться с цифры | ~~`2cats`~~ ❌ |
| без пробелов и спецсимволов | ~~`my var`~~ ❌ |
| нельзя использовать ключевые слова | ~~`int`, `class`~~ ❌ |
| **стиль camelCase** | `userName`, `totalSum` |

> Имя должно быть **осмысленным**: `userAge` лучше, чем `a`.

---

## Обмен значений двух переменных

Классический приём — через временную переменную:

```java
int a = 1, b = 2;
int temp = a;   // запомнили a
a = b;          // a = 2
b = temp;       // b = 1
// теперь a = 2, b = 1
```

## Подводные камни

| Ошибка | Объяснение |
|--------|-----------|
| `5 / 2 == 2` | целочисленное деление отбрасывает дробь |
| `int x = "5"` | нельзя присвоить строку в `int` — это разные типы |
| переполнение `int` | при выходе за диапазон значение «перескакивает» в минус |
| сравнение строк через `==` | для строк используют `.equals()` (см. [модуль 09](../m09_strings/theory.md)) |
