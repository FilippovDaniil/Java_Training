# Модуль 03. Типы данных и ввод с клавиатуры. Знакомство с IDEA

## Примитивные типы данных

В Java 8 **примитивных** типов. Они хранят само значение (не ссылку).

| Тип | Размер | Что хранит | Пример |
|------|--------|-----------|--------|
| `byte` | 1 байт | целое −128..127 | `byte b = 100;` |
| `short` | 2 байта | целое −32768..32767 | `short s = 5000;` |
| `int` | 4 байта | целое ±2,1 млрд | `int i = 100000;` |
| `long` | 8 байт | большое целое | `long l = 10_000_000_000L;` |
| `float` | 4 байта | дробное (7 знаков) | `float f = 3.14f;` |
| `double` | 8 байт | дробное (15 знаков) | `double d = 3.14159;` |
| `char` | 2 байта | один символ | `char c = 'A';` |
| `boolean` | 1 бит | `true` / `false` | `boolean ok = true;` |

> Суффиксы литералов: `L` для `long`, `f` для `float`, `d` для `double`.
> Символы — в **одинарных** кавычках: `'A'`. Строки — в двойных: `"A"`.

```
ЦЕЛЫЕ:    byte → short → int → long
ДРОБНЫЕ:  float → double
ЛОГИКА:   boolean
СИМВОЛ:   char
```

## Дробные числа и реальное деление

```java
double a = 7;
double b = 2;
System.out.println(a / b);   // 3.5  — деление double даёт дробь
System.out.println(7 / 2);   // 3    — а так int-деление (см. модуль 02)
```

> Чтобы получить дробный результат, хотя бы один операнд должен быть `double`/`float`.

## Приведение типов (casting)

### Автоматическое (расширяющее) — без потерь

```java
int x = 10;
double y = x;   // int → double автоматически: 10.0
```

### Явное (сужающее) — может терять данные

```java
double d = 9.99;
int i = (int) d;   // 9  — дробная часть отбрасывается

int code = 'A';        // 65 — char → int (код символа)
char ch = (char) 66;   // 'B' — int → char
```

```
расширение (авто):  byte → short → int → long → float → double
сужение (явно):     double → int  требует (int)
```

## Ввод с клавиатуры: `Scanner`

Класс `Scanner` читает данные из консоли. Его нужно **импортировать**.

```java
import java.util.Scanner;   // вверху файла, до class

public class Demo {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        System.out.print("Введите имя: ");
        String name = scanner.nextLine();    // строка целиком

        System.out.print("Введите возраст: ");
        int age = scanner.nextInt();         // целое число

        System.out.println("Привет, " + name + "! Тебе " + age);

        scanner.close();   // закрываем ресурс
    }
}
```

### Методы Scanner

| Метод | Что читает |
|-------|-----------|
| `nextInt()` | целое число (`int`) |
| `nextDouble()` | дробное число (`double`) |
| `nextLine()` | строку до конца строки |
| `next()` | одно слово (до пробела) |
| `nextBoolean()` | `true` / `false` |

> ⚠️ **Классическая ловушка:** после `nextInt()` в буфере остаётся `\n`. Если сразу вызвать `nextLine()`, он прочитает пустую строку. Решение — добавить лишний `scanner.nextLine();` после `nextInt()`.

## Подводные камни

| Проблема | Решение |
|----------|---------|
| `InputMismatchException` | ввели текст там, где ждали число |
| `nextLine()` читает пустоту после `nextInt()` | добавьте лишний `nextLine()` |
| потеря дробной части при `(int)` | используйте округление, если нужно |
| `long l = 10000000000;` ошибка | добавьте суффикс `L` |
