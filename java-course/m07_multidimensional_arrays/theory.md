# Модуль 07. Многомерные массивы

**Многомерный массив** — это «массив массивов». Чаще всего используется **двумерный** массив — таблица из строк и столбцов (матрица).

## Двумерный массив

```java
int[][] matrix = new int[3][4];   // 3 строки, 4 столбца
```

```
            столбцы →   0    1    2    3
          ┌──────────┬────┬────┬────┬────┐
строки ↓  │ строка 0 │ 0  │ 0  │ 0  │ 0  │
          ├──────────┼────┼────┼────┼────┤
          │ строка 1 │ 0  │ 0  │ 0  │ 0  │
          ├──────────┼────┼────┼────┼────┤
          │ строка 2 │ 0  │ 0  │ 0  │ 0  │
          └──────────┴────┴────┴────┴────┘

matrix[1][2] — элемент во 2-й строке, 3-м столбце
```

### Инициализация значениями

```java
int[][] m = {
    {1, 2, 3},
    {4, 5, 6},
    {7, 8, 9}
};
```

### Доступ к элементам и размеры

```java
m[0][0] = 10;                  // запись
System.out.println(m[2][1]);   // чтение → 8

System.out.println(m.length);      // 3  — количество строк
System.out.println(m[0].length);   // 3  — количество столбцов в строке 0
```

---

## Перебор двумерного массива

Нужны **вложенные циклы**: внешний — по строкам, внутренний — по столбцам.

```java
for (int row = 0; row < m.length; row++) {
    for (int col = 0; col < m[row].length; col++) {
        System.out.print(m[row][col] + "\t");
    }
    System.out.println();   // перевод строки после каждой строки матрицы
}
```

`for-each` тоже работает:

```java
for (int[] row : m) {          // row — это одномерный массив (строка)
    for (int value : row) {
        System.out.print(value + " ");
    }
    System.out.println();
}
```

## Типичные операции с матрицей

### Сумма всех элементов

```java
int sum = 0;
for (int[] row : m)
    for (int v : row)
        sum += v;
```

### Сумма по строке `i`

```java
int rowSum = 0;
for (int v : m[i]) rowSum += v;
```

### Главная диагональ (квадратная матрица)

```java
int diag = 0;
for (int i = 0; i < m.length; i++) {
    diag += m[i][i];   // элементы [0][0], [1][1], [2][2]...
}
```

## Зубчатые (jagged) массивы

Строки могут иметь **разную длину**:

```java
int[][] jagged = new int[3][];   // 3 строки, длины пока не заданы
jagged[0] = new int[]{1};
jagged[1] = new int[]{1, 2};
jagged[2] = new int[]{1, 2, 3};
```

```
строка 0: [1]
строка 1: [1, 2]
строка 2: [1, 2, 3]
```

> Поэтому всегда используйте `m[row].length` (длина конкретной строки), а не общую константу.

---

## Трёхмерные и выше

```java
int[][][] cube = new int[2][3][4];   // редко нужно, но возможно
```

> На практике почти всегда достаточно двумерных массивов.

---

## Удобный вывод: `Arrays.deepToString`

```java
import java.util.Arrays;
System.out.println(Arrays.deepToString(m));
// [[1, 2, 3], [4, 5, 6], [7, 8, 9]]
```

---

## Подводные камни

| Ошибка | Объяснение |
|--------|-----------|
| `m.length` как число столбцов | это число **строк**; столбцы — `m[i].length` |
| фиксированная длина строки для jagged | разные строки имеют разную длину |
| `Arrays.toString(m)` для 2D | напечатает адреса; нужно `deepToString` |
| перепутаны индексы `[row][col]` | первый индекс — строка, второй — столбец |
