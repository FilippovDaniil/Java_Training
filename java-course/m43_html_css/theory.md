# Модуль 43. HTML и CSS (факультатив)

> ⚠️ **Только теория, факультатив.** Это не Core Java, но базовое понимание HTML/CSS полезно Java-разработчику для веб-части.

**HTML** задаёт структуру и содержание веб-страницы, **CSS** — её внешний вид. Браузер читает HTML и CSS и рисует страницу.

```
HTML (структура)  +  CSS (оформление)  =  готовая страница
"что на странице"    "как это выглядит"
```

---

## Структура HTML-документа

```html
<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <title>Моя страница</title>
</head>
<body>
    <h1>Заголовок</h1>
    <p>Абзац текста.</p>
</body>
</html>
```

| Часть | Назначение |
|-------|-----------|
| `<!DOCTYPE html>` | версия HTML (HTML5) |
| `<head>` | метаданные (не видны на странице) |
| `<body>` | видимое содержимое |

---

## Основные теги HTML

```html
<h1>...</h1> ... <h6>...</h6>   <!-- заголовки -->
<p>Абзац</p>                     <!-- параграф -->
<a href="https://...">ссылка</a> <!-- ссылка -->
<img src="pic.jpg" alt="описание"> <!-- картинка -->
<ul><li>пункт</li></ul>          <!-- маркированный список -->
<ol><li>пункт</li></ol>          <!-- нумерованный список -->
<div>блок</div>                  <!-- блочный контейнер -->
<span>строчный</span>            <!-- строчный контейнер -->
<table><tr><td>ячейка</td></tr></table>  <!-- таблица -->
```

### Формы (ввод данных)

```html
<form action="/submit" method="post">
    <input type="text" name="username" placeholder="Имя">
    <input type="password" name="password">
    <button type="submit">Отправить</button>
</form>
```

> Форма отправляет данные на сервер (`action`) методом `method` — их обрабатывает сервлет (см. [модуль 39](../m39_servlets/theory.md)).

## CSS — оформление

### Способы подключения

```html
<!-- 1. внешний файл (рекомендуется) -->
<link rel="stylesheet" href="styles.css">

<!-- 2. внутренний -->
<style> p { color: blue; } </style>

<!-- 3. встроенный (избегать) -->
<p style="color: red;">текст</p>
```

### Синтаксис правила

```css
селектор {
    свойство: значение;
}

p {
    color: #333;
    font-size: 16px;
    margin: 10px;
}
```

## Селекторы CSS

```css
p          { }    /* по тегу */
.button    { }    /* по классу (class="button") */
#header    { }    /* по id (id="header") */
div p      { }    /* вложенные p внутри div */
a:hover    { }    /* псевдокласс (при наведении) */
```

| Селектор | Выбирает |
|----------|----------|
| `тег` | все такие теги |
| `.класс` | элементы с классом |
| `#id` | элемент с id (уникальный) |
| `a, b` | несколько селекторов |

## Блочная модель (box model)

Каждый элемент — это «коробка» из слоёв:

```
 +----------- margin (внешний отступ) -----------+
 |  +-------- border (рамка) ------------------+  |
 |  |  +----- padding (внутренний отступ) ---+  |  |
 |  |  |  +---- content (содержимое) ----+   |  |  |
 |  |  |  |       текст / картинка       |   |  |  |
 |  |  |  +------------------------------+   |  |  |
 |  |  +-------------------------------------+  |  |
 |  +--------------------------------------------+  |
 +-------------------------------------------------+
```

```css
.box {
    width: 200px;
    padding: 10px;     /* отступ внутри */
    border: 1px solid; /* рамка */
    margin: 20px;      /* отступ снаружи */
}
```

## Современная вёрстка (обзор)

| Технология | Назначение |
|------------|-----------|
| **Flexbox** | гибкое расположение в ряд/колонку |
| **Grid** | двумерные сетки |
| **Media queries** | адаптивность под экраны |
| **CSS-фреймворки** | Bootstrap, Tailwind — готовые стили |

```css
.container {
    display: flex;            /* flexbox */
    justify-content: center;
    gap: 16px;
}
```

## Связь с Java-разработкой

- Сервлеты/JSP **генерируют** HTML (модули 39, 41).
