# Модуль 05. Циклы: `while`, `for`, `do-while`, `break`, `continue`

**Цикл** повторяет блок кода, пока выполняется условие. Это избавляет от копирования одинаковых инструкций.

## Цикл `while` — «пока»

Проверяет условие **перед** каждой итерацией.

```java
int i = 1;
while (i <= 5) {       // пока i <= 5
    System.out.println(i);
    i++;               // ОБЯЗАТЕЛЬНО меняем i, иначе бесконечный цикл!
}
```

```
i=1 → печать → i=2 → печать → ... → i=6 → условие ложно → выход
```

> ⚠️ Если забыть `i++`, условие никогда не станет ложным — **бесконечный цикл**.

---

## Цикл `for` — когда известно число повторений

Объединяет три части в одной строке:

```java
//      инициализация; условие; шаг
for (int i = 1; i <= 5; i++) {
    System.out.println(i);
}
```

| Часть | Выполняется |
|-------|-------------|
| `int i = 1` | один раз в начале |
| `i <= 5` | перед каждой итерацией |
| `i++` | после каждой итерации |

Эквивалентен `while` выше, но компактнее. **Используйте `for`, когда число итераций известно.**

---

## Цикл `do-while` — «сделай хотя бы раз»

Проверяет условие **после** тела. Тело выполнится **минимум один раз**.

```java
int n;
do {
    System.out.print("Введите положительное число: ");
    n = scanner.nextInt();
} while (n <= 0);   // повторять, пока ввод некорректен
```

```
выполнить тело → проверить условие → если true, повторить
```

> Идеально для **валидации ввода** и меню — нужно показать хотя бы раз.

---

## Сравнение циклов

| Цикл | Проверка условия | Когда применять |
|------|------------------|-----------------|
| `for` | перед телом | известно число итераций |
| `while` | перед телом | число итераций неизвестно |
| `do-while` | после тела | тело должно выполниться ≥ 1 раза |

---

## `break` и `continue`

```java
for (int i = 1; i <= 10; i++) {
    if (i == 5) break;        // полностью прервать цикл
    if (i % 2 == 0) continue; // пропустить остаток итерации, перейти к следующей
    System.out.println(i);    // напечатает 1, 3
}
```

| Оператор | Действие |
|----------|----------|
| `break` | немедленно выйти из цикла |
| `continue` | пропустить оставшуюся часть тела, перейти к следующей итерации |

## Вложенные циклы

Цикл внутри цикла — например, для таблиц и двумерных структур.

```java
for (int row = 1; row <= 3; row++) {       // строки
    for (int col = 1; col <= 3; col++) {   // столбцы
        System.out.print(row * col + "\t");
    }
    System.out.println();   // перевод строки после каждой строки
}
```

Вывод (таблица умножения 3×3):

```
1   2   3
2   4   6
3   6   9
```

> Внутренний цикл выполняется **полностью** на каждой итерации внешнего.

---

## Накопление результата (аккумулятор)

Частый приём — переменная-накопитель **до** цикла:

```java
int sum = 0;                       // аккумулятор
for (int i = 1; i <= 100; i++) {
    sum += i;                      // sum = sum + i
}
System.out.println(sum);           // 5050
```

## Подводные камни

| Ошибка | Объяснение |
|--------|-----------|
| бесконечный цикл | забыли менять переменную условия |
| `for (int i = 0; i <= n; i++)` для массива | выход за границу (нужно `< n`, см. модуль 06) |
| объявление аккумулятора **внутри** цикла | он сбрасывается на каждой итерации |
| лишняя `;` после `for(...)` | `for(...);` — тело пустое, цикл «не работает» |
