# Задача 10 (мини-проект) — Модуль 07: Use Cases

## ЗАДАНИЕ
Постройте полную модель use cases для ключевого фрагмента FoodExpress: use case diagram + 3 детальных текстовых описания с альтернативами и исключениями. Синтез теории модуля.

## ВХОДНЫЕ ДАННЫЕ
Фрагмент FoodExpress: клиент находит ресторан, оформляет и оплачивает заказ, отслеживает доставку; ресторан подтверждает заказ; курьер доставляет. Используйте акторов и наработки задач 01–05.

## ТРЕБОВАНИЯ
Подготовьте модель, включающую:
1. **Use case diagram** — акторы (минимум Клиент, Менеджер ресторана, Курьер, Платёжный провайдер), use cases, границы, минимум 1 include и 1 extend.
2. **3 детальных текстовых описания** use case (например: «Оформить заказ», «Подтвердить заказ рестораном», «Отследить доставку») — каждое с полями: актор, цель, пред/постусловия, триггер, основной поток, альтернативы, исключения.
3. **Уровень** — все use cases sea-level.
4. **Трассировка** — каждый use case привязан к пользовательскому/бизнес-требованию.
5. **Согласованность** — пред/постусловия описаний согласованы между собой (цепочка состояний заказа).

## ФОРМАТ РЕЗУЛЬТАТА
Документ: диаграмма (псевдографика/PlantUML) + 3 текстовых use case + матрица трассировки. Внутренне согласованный.

## КРИТЕРИИ ПРИЁМКИ
- [ ] Диаграмма корректна по нотации, есть include и extend.
- [ ] 3 описания полны (все поля), с альтернативами и исключениями (не только happy path).
- [ ] Все use cases уровня sea-level.
- [ ] Пред/постусловия согласованы между описаниями (постусловие одного = предусловие другого).
- [ ] Каждый use case прослеживается до требования.
- [ ] Модель внутренне непротиворечива (статусы заказа согласуются между сценариями).
- [ ] Учтены минимум 2 исключения с реальными последствиями (отклонена оплата, нет товара/закрыт ресторан).

## ПОДСКАЗКА
Это «ядро» функциональной спецификации, которое дальше детализируется user stories (модуль 08), activity (13) и sequence (33). Главная проверка middle+ — полнота ветвлений и согласованность состояний между сценариями. Опирайтесь на задачи 01–05.
