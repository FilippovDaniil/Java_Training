# Задача 07 — Модуль 16: Модели данных и ER

## ЗАДАНИЕ
Изобразите готовую модель в нотации Crow's foot и прочитайте кратности по символам. Сопоставьте Crow's foot с записью UML-кратностей.

## ВХОДНЫЕ ДАННЫЕ
Связи FoodExpress:
1. Клиент — Заказ: у клиента 0+ заказов, у заказа ровно 1 клиент.
2. Заказ — Позиция: у заказа 1+ позиций, позиция принадлежит ровно 1 заказу.
3. Ресторан — Блюдо: у ресторана 1+ блюд, блюдо принадлежит ровно 1 ресторану.
4. Заказ — Курьер: у заказа 0..1 курьер, у курьера 0+ заказов.

## ТРЕБОВАНИЯ
- Для каждой связи изобразите Crow's foot (символы на обоих концах).
- Прочитайте связь словами в обе стороны.
- Сопоставьте каждую связь с UML-кратностями (`1`, `0..1`, `1..*`, `0..*`).
- Поясните значение символов (палочка/кружок/вилка).

## ФОРМАТ РЕЗУЛЬТАТА
Таблица `Связь | Crow's foot (символы) | Чтение словами | UML-кратность` + расшифровка символов.

## КРИТЕРИИ ПРИЁМКИ
- [ ] Для всех 4 связей показан Crow's foot на обоих концах.
- [ ] Кратности прочитаны словами в обе стороны.
- [ ] Сопоставление с UML-кратностями верное.
- [ ] Связь 2 и 3 — «обязательно один и более» (палочка + вилка).
- [ ] Связь 4 — «ноль или один» (кружок + палочка).
- [ ] Символы Crow's foot расшифрованы.

## ПОДСКАЗКА
Crow's foot: вилка («воронья лапка») = «много»; кружок = «ноль допустим»; палочка = «обязательно один». `─||─` ровно один, `─o|─` ноль/один, `─|<` один+, `─o<` ноль+. Соответствие UML: `─||─`↔`1`, `─o|─`↔`0..1`, `─|<`↔`1..*`, `─o<`↔`0..*`.
