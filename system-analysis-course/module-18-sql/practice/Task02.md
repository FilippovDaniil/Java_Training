# Задача 02 — Модуль 18: SQL

## ЗАДАНИЕ
Соедините две-три таблицы через JOIN.

## ВХОДНЫЕ ДАННЫЕ
Схема и данные — из `theory.md` модуля.

## ТРЕБОВАНИЯ
Выведите по каждому **доставленному** заказу: id заказа, имя клиента, название ресторана, сумму. Сортировка по id заказа.

## ФОРМАТ РЕЗУЛЬТАТА
SQL-запрос (INNER JOIN order + client + restaurant) + результат.

## ОЖИДАЕМЫЙ РЕЗУЛЬТАТ
```
100 | Иванов   | Суши-бар | 900
101 | Иванов   | Пиццерия | 600
103 | Сидорова | Суши-бар | 1500
104 | Сидорова | Пиццерия | 450
105 | Иванов   | Суши-бар | 300
```
(Заказ 102 — отменён, не выводится.)

## КРИТЕРИИ ПРИЁМКИ
- [ ] Используется INNER JOIN order↔client и order↔restaurant.
- [ ] Условие `status='Доставлен'`.
- [ ] 5 строк (без отменённого 102).
- [ ] Имена клиента и ресторана подставлены верно.
- [ ] Сортировка по id заказа.

## ПОДСКАЗКА
`FROM "order" o JOIN client c ON c.id=o.client_id JOIN restaurant r ON r.id=o.restaurant_id`. INNER JOIN берёт только совпадающие строки. Фильтр статуса — в WHERE.
