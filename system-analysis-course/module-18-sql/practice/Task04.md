# Задача 04 — Модуль 18: SQL

## ЗАДАНИЕ
Используйте подзапрос: найдите клиентов с заказом выше среднего чека.

## ВХОДНЫЕ ДАННЫЕ
Схема и данные — из `theory.md` модуля.

## ТРЕБОВАНИЯ
Найдите имена клиентов, у которых есть **доставленный** заказ с суммой выше средней суммы доставленного заказа. Средняя считается подзапросом.

## ФОРМАТ РЕЗУЛЬТАТА
SQL-запрос с подзапросом + результат + значение средней суммы.

## ОЖИДАЕМЫЙ РЕЗУЛЬТАТ
Средняя сумма доставленного заказа = (900+600+1500+450+300)/5 = **750**.
Заказы выше 750: 900 (Иванов), 1500 (Сидорова).
```
Иванов
Сидорова
```

## КРИТЕРИИ ПРИЁМКИ
- [ ] Подзапрос считает `AVG(total)` по доставленным (= 750).
- [ ] Внешний запрос отбирает заказы с `total > (подзапрос)`.
- [ ] Через JOIN/IN выведены имена клиентов.
- [ ] Результат: Иванов, Сидорова (без дублей).
- [ ] Отменённый заказ не влияет на среднее.

## ПОДСКАЗКА
`WHERE total > (SELECT AVG(total) FROM "order" WHERE status='Доставлен')`. Чтобы не было дублей клиента — `DISTINCT` или `IN (SELECT client_id ...)`. Подзапрос в скалярной позиции возвращает одно значение (среднее).
