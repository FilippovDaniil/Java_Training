# Задача 10 (мини-проект) — Модуль 35: Тестирование для аналитика

## ЗАДАНИЕ
Постройте тест-модель фичи: тест-кейсы из требований + тест-дизайн + Postman-проверки API + трассировка. Синтез теории модуля.

## ВХОДНЫЕ ДАННЫЕ
**FoodExpress**, фича «Оформление и оплата заказа»: создание заказа (мин. сумма 300 ₽, до 20 порций блюда), применение промокода, оплата через шлюз. API: `POST /orders`, `POST /orders/{id}/payment`. Требования имеют критерии приёмки.

## ТРЕБОВАНИЯ
Сдайте тест-модель:
1. **Перечень требований** фичи (функц.) с краткими критериями приёмки.
2. **Тест-кейсы** — позитивные + негативные, выведенные из требований (структура: предусловия/данные/шаги/ожидание).
3. **Тест-дизайн** — классы эквивалентности и граничные значения для правил (сумма, кол-во, промокод).
4. **Postman-проверки** API по контракту (статусы, схемы, ошибки RFC 7807) для обоих эндпоинтов.
5. **Трассировка** требование → тест-кейс (матрица покрытия, нет пробелов).
6. Краткая заметка: на каком уровне пирамиды что проверяется.

## ФОРМАТ РЕЗУЛЬТАТА
Структурированная тест-модель: требования+критерии, тест-кейсы (позитив/негатив), таблица тест-дизайна, Postman-проверки, матрица трассировки + 2 строки про уровни пирамиды.

## КРИТЕРИИ ПРИЁМКИ
- [ ] Перечень требований с критериями приёмки.
- [ ] Тест-кейсы со всеми полями; есть позитивные и негативные.
- [ ] Тест-дизайн: классы эквивалентности + границы для суммы/кол-ва/промокода.
- [ ] Postman-проверки для `POST /orders` и `POST /orders/{id}/payment` (статусы/схемы/ошибки).
- [ ] Матрица трассировки требование → тест без пробелов.
- [ ] Конкретные ожидаемые результаты во всех тестах.
- [ ] Отмечены уровни пирамиды (что unit/integration/API/UAT).

## ПОДСКАЗКА
Синтез: из UC/критериев → тест-кейсы (зад.01,08), тест-дизайн (02), структура кейса (03), позитив/негатив (05), Postman (06), трассировка (07), тест-данные (09). Главные проверки middle+: каждое требование покрыто (трассировка), есть негативы и границы, проверяемые ожидания, API сверен с контрактом. Это вход в поставку фичи (модуль 36).
