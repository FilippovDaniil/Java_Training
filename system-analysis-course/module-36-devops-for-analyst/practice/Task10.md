# Задача 10 (мини-проект) — Модуль 36: DevOps для аналитика

## ЗАДАНИЕ
Составьте план поставки фичи: среды, конфигурация, миграция, стратегия выката, флаги, откат, мониторинг. Синтез теории модуля.

## ВХОДНЫЕ ДАННЫЕ
**FoodExpress**, фича «подписка на доставку» (платная подписка с автосписанием). Требует: новые поля/таблицы в БД, интеграцию с платёжным шлюзом, постепенный выкат на аудиторию, возможность мгновенно выключить. Сервис работает 24/7.

## ТРЕБОВАНИЯ
Сдайте план поставки:
1. **Промоушн по средам** — путь dev→test→stage→prod, что проверяется на каждой (включая UAT).
2. **CI/CD** — ключевые этапы и где проходят тесты/критерии приёмки.
3. **Миграция БД** — план expand/contract для новых полей/таблиц (без даунтайма).
4. **Конфигурация/секреты** — что отличается по средам (платёжный шлюз тест/прод, ключи).
5. **Стратегия выката** — canary/blue-green + обоснование; постепенный выкат через feature flag.
6. **План отката** — как откатить (флаг/среда/образ) и по каким критериям.
7. **Мониторинг** — какие метрики/сигналы смотреть после выката (бизнес + технические).
8. **Чек-лист готовности** к релизу.
Короткий вывод: как план обеспечивает безопасную поставку с возможностью отката.

## ФОРМАТ РЕЗУЛЬТАТА
Структурированный план поставки (среды, CI/CD, миграция, конфиг, выкат+флаг, откат, мониторинг, чек-лист) + 2–3 строки вывода.

## КРИТЕРИИ ПРИЁМКИ
- [ ] Промоушн по средам с проверками и UAT.
- [ ] Этапы CI/CD с местом тестов/критериев приёмки.
- [ ] Миграция expand/contract для новых полей/таблиц (без даунтайма).
- [ ] Конфигурация/секреты по средам (платёжный шлюз тест/прод).
- [ ] Стратегия выката (canary/blue-green) + feature flag для постепенного включения.
- [ ] План отката с критериями (флаг/среда/образ).
- [ ] Метрики мониторинга (бизнес + технические).
- [ ] Чек-лист готовности к релизу.
- [ ] Вывод о безопасности поставки и откате.

## ПОДСКАЗКА

Главные проверки middle+: поставка без даунтайма (expand/contract), постепенный выкат с мгновенным откатом (флаг/blue-green), разделение деплой/релиз, мониторинг бизнес-метрик.
