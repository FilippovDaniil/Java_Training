# Задача 08 — Модуль 37: Эксплуатация и сопровождение

## ЗАДАНИЕ
Проведите оценку результатов внедрения фичи по принципу «до/после».

## ВХОДНЫЕ ДАННЫЕ
**FoodExpress**, три месяца назад внедрена фича «подписка на доставку» с бизнес-целью: повысить долю повторных заказов и LTV (модуль 03). Нужно оценить, достигнута ли цель.

## ТРЕБОВАНИЯ
- Определите метрики оценки (привязанные к цели фичи): целевые и поддерживающие.
- Опишите измерение «до/после» (база сравнения, период, сегменты).
- Сформулируйте критерий успеха (что считать достижением цели).
- Опишите возможные решения по итогам: развивать / докрутить / откатить.
- Отметьте риски интерпретации (сезонность, другие факторы — корреляция ≠ причинность).

## ФОРМАТ РЕЗУЛЬТАТА
Набор метрик + схема измерения до/после + критерий успеха + варианты решений + 1–2 строки про риски интерпретации.

## КРИТЕРИИ ПРИЁМКИ
- [ ] Метрики привязаны к бизнес-цели фичи (повторные заказы, LTV, retention).
- [ ] Описано измерение до/после (база, период, сегменты).
- [ ] Сформулирован критерий успеха.
- [ ] Предложены решения по итогам (развивать/докрутить/откатить).
- [ ] Учтены риски интерпретации (сезонность, иные факторы).

## ПОДСКАЗКА
Опора — §6 теории и модуль 03. Оценка замыкает цикл: фичу делали ради цели → проверить по метрикам до/после. Осторожно с выводами: учитывай сезонность и другие изменения (корреляция ≠ причинность).
