# Задача 09 — Модуль 37: Эксплуатация и сопровождение

## ЗАДАНИЕ
Составьте план обучения пользователей при запуске системы.

## ВХОДНЫЕ ДАННЫЕ
**MedConnect** запускается для двух групп пользователей: врачи (внутренние, работают ежедневно) и пациенты (внешние, эпизодически). Система заменяет привычные процессы.

## ТРЕБОВАНИЯ
- Спроектируйте план обучения под каждую группу (форматы, объём, материалы).
- Учтите разницу аудиторий: врачи (глубоко, регулярно) vs пациенты (просто, самостоятельно).
- Подберите форматы: очные/вебинары, видео, инструкции, FAQ, onboarding в интерфейсе.
- Свяжите с документацией (руководство пользователя, задача 04) и поддержкой (L1).
- Отметьте, как измерить успешность обучения (адаптация, обращения в поддержку).

## ФОРМАТ РЕЗУЛЬТАТА
План обучения по группам (формат, материалы, объём) + связь с документацией/поддержкой + метрики успешности.

## КРИТЕРИИ ПРИЁМКИ
- [ ] План учитывает обе группы и их различия.
- [ ] Подобраны адекватные форматы под каждую аудиторию.
- [ ] Связь с руководством пользователя и поддержкой L1.
- [ ] Учтено сопротивление изменениям (замена привычных процессов).
- [ ] Метрики успешности обучения (адаптация, снижение обращений).

## ПОДСКАЗКА
Опора — §1, §4, §6 теории. Врачам — глубокое регулярное обучение (вебинары, инструкции); пациентам — простой onboarding и FAQ. Обучение опирается на руководство пользователя и снижает нагрузку на L1. Успех мерить по адаптации/обращениям.
