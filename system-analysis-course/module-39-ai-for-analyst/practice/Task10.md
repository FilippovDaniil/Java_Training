# Задача 10 (мини-проект) — Модуль 39: ИИ для системного аналитика

## ЗАДАНИЕ
Постройте связку «требование → ИИ-черновики артефактов → верификация» на сквозном кейсе. Синтез модуля.

## ВХОДНЫЕ ДАННЫЕ
**Умный дом / IoT** (capstone-домен, `C`), фича «Сценарий автоматизации»: пользователь создаёт правило «если датчик движения сработал ночью → включить свет». Нужно с помощью ИИ быстро подготовить пакет черновиков и верифицировать их.

## ТРЕБОВАНИЯ
Пройдите полный цикл с ИИ под верификацией:
1. **Промпты** для генерации (с ролью/контекстом/форматом/ограничениями) — для каждого артефакта.
2. **Черновики от ИИ:** (а) user stories + критерии приёмки; (б) фрагмент модели данных/SQL для хранения правил; (в) диаграмма сценария (sequence/BPMN-текст); (г) тест-кейсы по критериям.
3. **Верификация каждого артефакта:** что проверить и как (INVEST/Gherkin, схема/прогон SQL, нотация/логика диаграммы, трассировка покрытия тестов).
4. **Конфиденциальность:** что обезличить перед отправкой (данные о присутствии людей — ПДн).
5. **Найденные дефекты ИИ:** минимум 3 (галлюцинация/пропуск/неполнота) + исправления.
6. **Вывод:** что ИИ ускорил, что осталось за аналитиком, где ответственность.

## ФОРМАТ РЕЗУЛЬТАТА
Структурированный пакет: промпты + 4 черновика артефактов + верификация каждого + заметка по конфиденциальности + список дефектов ИИ с правками + вывод.

## КРИТЕРИИ ПРИЁМКИ
- [ ] Промпты со всеми элементами (роль/контекст/формат/ограничения).
- [ ] 4 черновика артефактов (US+критерии, данные/SQL, диаграмма, тест-кейсы).
- [ ] Верификация каждого артефакта (конкретный способ проверки).
- [ ] Учтена конфиденциальность (ПДн о присутствии → обезличить).
- [ ] ≥3 найденных дефекта ИИ с исправлениями.
- [ ] Артефакты согласованы между собой (один сценарий).
- [ ] Вывод: что ускорено, что за аналитиком, где ответственность.

## ПОДСКАЗКА
Синтез всего модуля: промпт (зад.01), SQL+проверка (02), US+ревью (04), диаграмма+проверка (05), тесты+покрытие (07), галлюцинации (08), конфиденциальность (09). Главная мысль middle+ и финал курса: **ИИ даёт черновики быстро, но истина, полнота, безопасность и ответственность — за аналитиком**. Покажи полный цикл «генерация → верификация → ответственность» — это и есть зрелое применение ИИ.
