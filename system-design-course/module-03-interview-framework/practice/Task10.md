# Задача 10 — Модуль 03 (мини-проект): Полный план ответа на design-интервью

## ЗАДАНИЕ
Составьте полный тезисный план ответа на design-интервью по заданной системе, пройдя все 8 шагов фреймворка с тайм-боксом. Это синтез модуля: не полный дизайн, а «партитура» вашего выступления.

## ВХОДНЫЕ ДАННЫЕ
Система на выбор (зафиксируйте одну): (а) сервис коротких ссылок; (б) система уведомлений; (в) лента соцсети. Недостающие требования закройте явными допущениями.

## ТРЕБОВАНИЯ
План должен пройти все 8 шагов, по каждому — тезисы (а не полное решение):
1. **Требования и scope** — 4–6 уточняющих вопросов + зафиксированные допущения + что вне границ.
2. **Estimation** — 2–3 ключевых числа (RPS read/write, объём) с прикидкой.
3. **API** — 3–5 ключевых эндпоинтов/методов.
4. **Data model** — ключевые сущности.
5. **High-Level Design** — список блоков + поток чтения и поток записи (тезисно).
6. **Deep dive** — какой компонент углубили бы и какие вопросы раскрыли.
7. **Bottlenecks & scaling** — 2–3 узких места + что при отказе + named trade-offs.
8. **Wrap-up** — резюме решений + что улучшить дальше.
- Проставьте ориентировочный тайминг по шагам (сумма ≈ 40 мин).

## ФОРМАТ РЕЗУЛЬТАТА
Структурированный план из 8 пронумерованных шагов с тезисами и таймингом.

## КРИТЕРИИ ПРИЁМКИ
- [ ] Все 8 шагов присутствуют в правильном порядке, с таймингом (HLD — наибольшее время).
- [ ] Шаг 1 содержит уточняющие вопросы и явные допущения; обозначен scope.
- [ ] Шаг 2 содержит конкретные числа с прикидкой (read/write-ratio выявлен).
- [ ] Шаг 5 разделяет поток чтения и поток записи.
- [ ] Шаг 6 выбирает архитектурно-богатый компонент для углубления.
- [ ] Шаг 7 называет узкие места, поведение при отказе и хотя бы один trade-off.
- [ ] Нет преждевременной сложности; решения соразмерны числам из шага 2.

## ПОДСКАЗКА
Это репетиция реального интервью «на бумаге». Цель — отработать переходы между шагами и тайминг, чтобы на живом интервью фреймворк работал автоматически. Связность обязательна: блоки в шаге 5 должны следовать из чисел шага 2, а узкие места шага 7 — из этих же блоков. Этот план — мост к Части 8, где те же системы проектируются полностью.
