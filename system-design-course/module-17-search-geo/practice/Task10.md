# Задача 10 — Модуль 17 (мини-проект): Поиск + автодополнение + гео

## ЗАДАНИЕ
Спроектируйте подсистему поиска, автодополнения и геопоиска для системы. Синтез модуля и завершение Части 3.

## ВХОДНЫЕ ДАННЫЕ
Система на выбор (зафиксируйте одну): (а) поиск ресторанов рядом (доставка); (б) поиск товаров в маркетплейсе с подсказками; (в) поиск мест/событий на карте. Объёмы и нагрузку возьмите из расчётов модулей 05–07 (или задайте явно как допущение).

## ТРЕБОВАНИЯ
Дизайн должен покрывать:
1. **Полнотекстовый поиск** — inverted index/движок; конвейер обработки текста; ранжирование.
2. **Индексация** — синхронизация индекса с БД (CDC/очередь), lag, перестроение; индекс ≠ источник правды.
3. **Автодополнение** — trie + предрасчёт топ-K + кэш популярных префиксов.
4. **Геопоиск** — выбор структуры (geohash/quadtree/S2), nearby-запрос, граничная проблема.
5. **Шардирование/хранение** — где живут индекс и гео-данные; hot zone.
6. **Кэширование** — что и где кэшируем (популярные запросы/префиксы/ячейки).
7. **Схема** — диаграмма потоков (БД → индексация → поиск/гео/typeahead → пользователь).

## ФОРМАТ РЕЗУЛЬТАТА
Структурированный дизайн-документ из 7 разделов + диаграмма.

## КРИТЕРИИ ПРИЁМКИ
- [ ] Зафиксирована одна система; объёмы/нагрузка заданы (своя оценка или допущение).
- [ ] Поиск через inverted index/движок с конвейером и ранжированием (не LIKE).
- [ ] Индексация через CDC/очередь; индекс eventually consistent, не источник правды; есть перестроение.
- [ ] Автодополнение через trie + предрасчёт топ-K + кэш.
- [ ] Геопоиск через пространственную структуру с обработкой граничной проблемы.
- [ ] Решены шардирование/хранение и hot zone; кэш популярного.
- [ ] Есть схема потоков; решения опираются на числа Части 2 и связаны с модулями 09/13/15.

## ПОДСКАЗКА
Это синтез модуля и финал Части 3. Все три блока объединяет идея: специальная задача → специальная структура (inverted index/trie/geohash) + отдельное eventually consistent хранилище, синхронизируемое с источником правды через события (модуль 15), с кэшем популярного (модуль 09) и шардированием (модуль 13).
