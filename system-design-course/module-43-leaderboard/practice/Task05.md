# Задача 05 — Модуль 43: Запросы лидерборда

## ЗАДАНИЕ
Реализуйте три типичных запроса лидерборда через команды sorted set и обработайте ties.

## ВХОДНЫЕ ДАННЫЕ
UI показывает: топ-10 лидеров, «вы #1234», и «соседи по рейтингу» (ближайшие к пользователю). Бывают равные счета (ties).

## ТРЕБОВАНИЯ
- Top-K: ZREVRANGE 0 K-1.
- Ранг пользователя: ZREVRANK.
- Диапазон вокруг пользователя: ранг → ZREVRANGE rank-k rank+k.
- Обработка ties (равные счета): вторичный критерий (например, время в score).

## ФОРМАТ РЕЗУЛЬТАТА
Top-K → ранг → диапазон вокруг → обработка ties.

## КРИТЕРИИ ПРИЁМКИ
- [ ] Top-K: ZREVRANGE board 0 K-1 (WITHSCORES).
- [ ] Ранг: ZREVRANK board user.
- [ ] Вокруг пользователя: получить ранг → ZREVRANGE rank-k rank+k.
- [ ] Ties: вторичный критерий (время достижения счёта, закодированное в score) для однозначного порядка.

## ПОДСКАЗКА
Top-K: ZREVRANGE 0 K-1. Ранг: ZREVRANK. Вокруг: rank=ZREVRANK → ZREVRANGE rank±k. Ties: вторичный критерий (время) в score.
