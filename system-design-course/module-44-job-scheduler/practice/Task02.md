# Задача 02 — Модуль 44: Estimation

## ЗАДАНИЕ
Оцените нагрузку планировщика и покажите неравномерность срабатываний во времени.

## ВХОДНЫЕ ДАННЫЕ
Допущения (примите явно): млн запланированных задач, распределение времён срабатывания (много на полночь/начало часа), производительность воркера, storage определений/истории.

## ТРЕБОВАНИЯ
- Оцените число запланированных задач и storage (определения + история).
- Покажите неравномерность срабатываний/с (пики в «круглые» времена vs среднее).
- Оцените число воркеров под пиковую нагрузку срабатываний.
- Сделайте вывод: проектировать под пик (thundering herd), не под среднее.

## ФОРМАТ РЕЗУЛЬТАТА
Допущения → задачи/storage → пик vs среднее срабатываний → число воркеров → вывод про пик.

## КРИТЕРИИ ПРИЁМКИ
- [ ] Число задач и storage (определения + история) оценены.
- [ ] Показана неравномерность: пик срабатываний (полночь) ≫ среднее.
- [ ] Число воркеров = пиковая нагрузка / производительность воркера.
- [ ] Вывод: проектировать под пик (thundering herd), сглаживать джиттером.

## ПОДСКАЗКА
Млн задач + история → storage. Срабатывания неравномерны: пик в полночь ≫ среднее. Воркеры = пик / производительность. Проектировать под пик, сглаживать джиттером.
