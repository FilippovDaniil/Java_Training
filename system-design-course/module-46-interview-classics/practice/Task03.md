# Задача 03 — Модуль 46: Распознавание незнакомой задачи

## ЗАДАНИЕ
Сведите незнакомую задачу к известным архетипам и подберите блоки.

## ВХОДНЫЕ ДАННЫЕ
Незнакомые постановки (нет в каталоге): (а) «система бронирования переговорных комнат», (б) «система антифрода для транзакций», (в) «коллаборативный редактор документов».

## ТРЕБОВАНИЯ
- Для каждой задачи уточните доминирующую ось (через предполагаемые FR/NFR).
- Сведите к архетипу(ам).
- Подберите блоки по карте (§3) для этих архетипов.
- Зафиксируйте ключевые допущения.

## ФОРМАТ РЕЗУЛЬТАТА
Для каждой задачи: ось → архетип(ы) → блоки → допущения.

## КРИТЕРИИ ПРИЁМКИ
- [ ] Бронирование → correctness-critical (нет двойного бронирования) → идемпотентность + ACID/locking (23/21).
- [ ] Антифрод → write-heavy ingestion + real-time scoring → очередь + стрим + правила/ML.
- [ ] Коллаборативный редактор → real-time + конфликты → WebSocket (35) + CRDT/OT (12).
- [ ] Для каждой зафиксированы явные допущения; показан метод сведения к архетипу.

## ПОДСКАЗКА
Бронирование = correctness (нет двойного бронирования) → ACID/locking+идемпотентность. Антифрод = write-heavy+real-time → очередь+стрим-scoring. Редактор = real-time+конфликты → WebSocket+CRDT. Метод: ось→архетип→блоки→допущения.
