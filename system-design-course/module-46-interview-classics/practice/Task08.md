## ТРЕБОВАНИЯ
- Найдите минимум 5 ошибок подхода.
- Для каждой объясните, почему это плохо и как правильно.
- Свяжите с методикой (8 шагов, think aloud, архетип, числа).
- Покажите правильную последовательность для этой задачи.

## ФОРМАТ РЕЗУЛЬТАТА
Список ошибок → почему плохо → как правильно → корректная последовательность.

- [ ] Для каждой — почему плохо и как правильно.
