# Задача 10 — Модуль 46 (мини-проект): Персональный interview playbook

## ЗАДАНИЕ
Составьте персональный interview playbook — компактный справочник для ответа на любую SD-задачу.

## ВХОДНЫЕ ДАННЫЕ
Нужен один документ, который вы открываете перед интервью: архетипы, карта блоков, методика, шпаргалка. Синтез модуля.

## ТРЕБОВАНИЯ
Playbook из разделов:
1. **Архетипы** — список с признаками распознавания.
2. **Карта паттернов** — архетип → строительные блоки (с модулями).
3. **Методика** — 8 шагов + think aloud + тайм-менеджмент.
4. **Распознавание незнакомой задачи** — алгоритм сведения к архетипу.
5. **Шпаргалка** — latency numbers, порядки, peak factor, типовые развилки.
6. **Каталог** — ≈20 классических задач по архетипам.

## ФОРМАТ РЕЗУЛЬТАТА
Playbook из 6 разделов (компактно, как справочник для открытия перед интервью).

## КРИТЕРИИ ПРИЁМКИ
- [ ] Архетипы с признаками распознавания (read-heavy/write-heavy/real-time/geo/storage/correctness/search/coordination).
- [ ] Карта паттернов архетип → блоки (с модулями Частей 3–7).
- [ ] Методика: 8 шагов + think aloud + тайм-менеджмент (модуль 03).
- [ ] Алгоритм распознавания незнакомой задачи (требования → архетип → блоки → допущения).
- [ ] Шпаргалка (latency numbers, порядки, peak factor, типовые развилки: 301/302, push/pull, sync/async, SQL/NoSQL).
- [ ] Каталог ≈20 задач по архетипам (связь с Частью 8).

## ПОДСКАЗКА
6 разделов: архетипы (признаки) → карта блоков → методика (8 шагов) → распознавание незнакомой (ось→архетип→блоки→допущения) → шпаргалка (числа+развилки) → каталог ≈20 задач. Это синтез модуля.
