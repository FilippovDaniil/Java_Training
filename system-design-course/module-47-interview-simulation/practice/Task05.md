# Задача 05 — Модуль 47: Сигналы интервьюера

## ЗАДАНИЕ
Перечислите сигналы, которые ищет интервьюер, и покажите, как их продемонстрировать.

## ВХОДНЫЕ ДАННЫЕ
Интервью оценивает не «правильный ответ», а сигналы мышления. Нужно сознательно их подавать.

## ТРЕБОВАНИЯ
- Перечислите ключевые сигналы (структурное мышление, requirements first, квантификация, trade-offs, глубина, коммуникация, реакция на подсказки, работа с неоднозначностью).
- Для каждого покажите, как продемонстрировать на практике.
- Свяжите сигналы с рубрикой (какие поднимают до senior).
- Объясните, почему явная подача сигналов важнее «идеальной схемы».

## ФОРМАТ РЕЗУЛЬТАТА
Сигнал → как продемонстрировать → связь с грейдом → почему сигналы важнее схемы.

## КРИТЕРИИ ПРИЁМКИ
- [ ] Перечислены сигналы: структурное мышление, requirements first, квантификация, trade-offs, глубина, коммуникация, реакция на подсказки, работа с неоднозначностью.
- [ ] Для каждого — способ демонстрации (называть фазы, уточнять, считать, проговаривать альтернативы, углубляться).
- [ ] Сигналы связаны с грейдом (квантификация/trade-offs/глубина → senior).
- [ ] Объяснено: интервью = сбор сигналов, поэтому явная подача важнее «идеальной» схемы.

## ПОДСКАЗКА
Сигналы: структура, requirements first, квантификация, trade-offs, глубина, коммуникация, реакция на подсказки, неоднозначность. Демонстрировать явно (называть фазы, считать, проговаривать альтернативы). Квантификация/trade-offs/глубина → senior.
