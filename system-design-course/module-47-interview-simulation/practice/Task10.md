# Задача 10 — Модуль 47 (мини-проект): Полная симуляция + самооценка

## ЗАДАНИЕ
Проведите полную симуляцию интервью на классической задаче и оцените себя по рубрике. Синтез модуля и завершение Части 9.

## ВХОДНЫЕ ДАННЫЕ
Выберите классическую задачу (каталог модуля 46). 60 минут. Цель — пройти все фазы, продемонстрировать сигналы, затем измеримо оценить себя.

## ТРЕБОВАНИЯ
1. **Симуляция** — полный ход ответа по фазам с таймингом (требования→...→wrap-up), think-aloud, ≥2 trade-offs вслух, deep dive в 1–2 компонента.
2. **Сигналы** — отметьте, где продемонстрировали каждый ключевой сигнал.
3. **Самооценка** — оцените ответ по рубрике (грейд + обоснование).
4. **План улучшения** — 1–2 слабых сигнала для тренировки.

## ФОРМАТ РЕЗУЛЬТАТА
Симуляция (фазы+тайминг) → карта сигналов → грейд по рубрике + обоснование → план улучшения.

## КРИТЕРИИ ПРИЁМКИ
- [ ] Симуляция: все фазы с таймингом в 60 мин; think-aloud; ≥2 trade-offs вслух; deep dive в 1–2 компонента.
- [ ] Архетип задачи назван, блоки подобраны (связь с модулем 46).
- [ ] Карта сигналов: где продемонстрированы структура/requirements first/квантификация/trade-offs/глубина/коммуникация.
- [ ] Самооценка по рубрике: грейд + обоснование конкретными сигналами.
- [ ] План улучшения: 1–2 слабых сигнала для целенаправленной тренировки.

## ПОДСКАЗКА
Выбери задачу из каталога (модуль 46) → симуляция по фазам с таймингом (think-aloud, trade-offs, deep dive) → отметь поданные сигналы → оцени грейд по рубрике с обоснованием → выбери 1–2 слабых сигнала для тренировки. Завершает Часть 9.
