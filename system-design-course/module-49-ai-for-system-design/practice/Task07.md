ИИ-фича работает на GPU и/или платном LLM API. Стоимость inference растёт с нагрузкой и длиной промптов и может доминировать в TCO.

## ТРЕБОВАНИЯ

- Предложите оптимизации: батчинг, кэш ответов, меньшая модель/квантизация, prompt caching, RAG вместо больших промптов.
- Сравните с другими статьями cost (compute/storage/egress — модуль 7).

## ФОРМАТ РЕЗУЛЬТАТА
